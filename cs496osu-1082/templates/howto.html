How-to use Jenkins with web and windows apps

This tutorial will cover using Jenkins CI as it's intended, for continuous integration. Specifically, it will cover getting a cloud based Google App Engine website deployed, windows app compiled, and all of them tested. From installation to automatic build and deploy off of a github code change.

This will be used to ensure that the development is the only thing to worry about and new tests and code can be added with very little extra effort. This is the core idea behind continuous integration and this will be the focus of this how-to.
