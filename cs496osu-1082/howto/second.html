Github Plugin

The next step will be to install the github plugin so that code changes can kick off builds. There is one caveat, you need a HTTP URL reachable from GitHub. What that means is either your computer, if you are running locally, or the server you have installed Jenkins to need at a minimum a static ip address and http properly routed. Once you have that come back to this page.

First you will got to Manage Jenkins, then Manage Plugins, select the available tab and filter based off of github plugin and click the install box and then Download now and install after restart. It will then go through a few pages where it installs the plugins and click restart Jenkins automatically when complete.
