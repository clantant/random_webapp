Installing Jenkins

First you will need to install Jenkins CI for your platform of choice, there isn't much here that isn't covered fully in the original documentation here: Installing Jenkins

The main thing Jenkins needs is Java greater than version 7, it's that simple dependency that makes it so extensible. What this means is that you can put it pretty much anywhere. That feature is especially important for the deployment phase and for parallelizing the build and test process.
